欢迎使用象限时光

象限时光（XXSG） 是一个基于艾森豪威尔矩阵的本地优先（local-first）PWA 生产力工具。 它把四象限时间管理、习惯打卡、番茄专注、日程管理、AI 分析整合在一起， 帮你从混乱中找到秩序，把精力花在真正重要的事情上。

数据默认存储在浏览器本地，无需注册即可使用。 登录后可自定义设置，AI 功能需自行配置 API Key。

本地优先 离线可用 数据自主 PWA 可安装

核心功能

四象限任务

按重要×紧急自动分类到四个象限，拖拽切换优先级，开始/截止时间管理，置顶、子任务、双视图（列表/网格）一应俱全。

⏰ 智能提醒系统

开始时间前 5 分钟弹窗提醒；时间一到任务卡片自动变红（左边框 + 渐变背景 + 红色标题闪动）；手动关闭后不再打扰。到期、过期均有提醒。

日历日程

月视图管理日程事件，支持从四象限一键同步任务到日历，按分类/状态/象限筛选。

习惯打卡

自定义习惯类型、图标和备注，每日打卡记录连续天数和完成率，成就徽章激励，打卡趋势图表。

番茄专注

25 分钟工作 + 5 分钟休息的标准 Pomodoro，支持自定义时长、完成音效、历史记录统计。

数据看板

总任务数、完成率、象限分布、趋势图表（饼图/柱状图/折线图），实时更新，一目了然。

AI 智能分析

接入 DeepSeek / OpenAI API（需自行配置 Key），自动分析任务属性、推荐象限、估算时间、生成每日签语。

顶级复盘

日/周/月/年四级复盘体系，记录成长历程，总结经验教训，形成持续改进的闭环。

倒数纪念日

管理生日、纪念日、重要事件等多种类型的日期倒数，智能提醒不错过重要日子。

时间看板

记录每日时间分配，饼图/柱状图/趋势图可视化分析，帮你发现时间黑洞，优化分配策略。

自然日历

山景云海的沉浸式氛围中规划日程，支持周视图、任务视图切换，让日程管理成为享受。

每日一签

传统签语 + AI 智能生成两种模式，每日抽取幸运签，为一天带来正能量和灵感。

数据备份

支持 JSON 导出/导入，写入前自动备份旧数据到 localStorage，避免误操作丢失。

主题切换

浅色/深色主题随意切换，毛玻璃效果 + 现代化设计语言，护眼又美观。

四象限方法论

艾森豪威尔矩阵 — 按「重要性」和「紧急性」两个维度把任务分为四类

🔴

重要且紧急

危机处理 · 立即行动

紧急会议、突发问题、截止日期迫近的任务。

尽量减少此类任务 — 通过提前规划把更多事情做到「重要不紧急」
🟡

重要不紧急

规划发展 · 重点投入

学习成长、健康管理、长期目标、关系维护。

这是最值得投入时间的象限，决定未来的成就
🔵

不重要但紧急

日常事务 · 委托/批量

部分电话/邮件、临时会议、他人请求。

尽量委托他人或集中批量处理，不要占据核心时间
🟢

不重要不紧急

休闲娱乐 · 适度节制

无目的刷手机、闲聊、过度娱乐。

适度放松有必要，但不要让它成为时间的主要去向

快速上手

1

创建任务

在「添加任务」标签页填写标题、选择优先级（1–4 对应四个象限）、设置开始/截止时间。保存后自动归类到对应象限。

2

四象限视图

切换到「四象限视图」查看任务的全部分布。拖拽任务可以跨象限移动（改变优先级），点击标题编辑详情。

3

标记完成

点击任务左侧的圆圈按钮切换完成状态。完成的任务自动从象限中隐藏（可在看板中查看）。

4

编辑与详情

点击任务标题打开详情弹窗，可编辑所有字段、添加子任务、复制、删除、置顶。弹窗支持拖拽移动。

5

提醒与过期

设置了开始时间的任务，到期前 5 分钟弹窗提醒。时间已过的任务卡片变红色 + 红色边框。关闭后不再重复打扰。

6

视图切换

四象限支持「网格模式」和「列表模式」一键切换。右上角按钮可展开单个象限全屏查看。

7

数据导出/导入

在设置或数据同步页面可导出 JSON 备份，或从备份文件恢复数据。切换浏览器前务必先导出。

8

AI 配置

进入设置 → AI 服务，填入 DeepSeek 或 OpenAI 的 API Key 和基础 URL，即可使用智能分析功能。

提醒系统说明

⏰ 开始时间提醒

任务设置了开始时间后：

  • 到期前 5 分钟（可配置）弹窗提醒
  • 时间已过的任务卡片自动变 红色（左边框 + 渐变背景 + 红色标题）
  • 手动关闭弹窗后不再重复弹出
  • 错过时间的任务，下次打开页面时也会触发一次

⏳ 截止时间提醒

任务设置了截止时间后：

  • 到期前 30 分钟内触发弹窗提醒
  • 已过期的截止时间文字变红色闪动
  • 支持浏览器桌面通知（需授权）
  • 提醒记录保留 7 天，7 天后自动清理

⚙️ 配置方式

  • 提前提醒时间可在设置中调整
  • 通知权限在「通知设置」中申请
  • 音效/震动可在设置中开关
  • 浏览器控制台执行 window.checkReminderSystemStatus() 诊断

常见问题

Q1

数据会丢失吗？

A： 数据保存在浏览器 localStorage 中。清除浏览器数据、更换浏览器、重装系统 会导致数据丢失。务必定期导出 JSON 备份。

Q2

支持多设备同步吗？

A： 当前版本为本地优先（local-first），数据仅在浏览器本地。多设备同步需要 SaaS 后端（开发中）。目前可以通过导出 JSON → 另一设备导入的方式手动迁移。

Q3

AI 功能怎么用？

A： 设置 → AI 服务中填入 DeepSeek 或 OpenAI 的 API Key（需自行注册账号获取）。AI 分析调用第三方 API 会产生费用，由对应平台收取。

Q4

如何备份数据？

A： Settings → 数据导出 → 选择 JSON 格式下载到本地。恢复时在数据同步页面上传备份文件。建议每次重大操作前导出一次。

Q5

支持离线使用吗？

A： 基础功能（任务/习惯/番茄钟/看板）离线可用。AI 分析需要网络。安装 PWA 到桌面后离线体验更佳。

Q6

弹窗提醒不弹出？

A： ① 检查「通知设置」是否开启了桌面通知权限；② 提醒系统每 2 秒轮询一次，页面在后台时浏览器会节流；③ 控制台执行 window.checkReminderSystemStatus() 诊断。

Q7

如何清除所有数据？

A： 设置 → 数据管理 → 清空所有数据。⚠️ 此操作不可恢复，请先导出备份。
