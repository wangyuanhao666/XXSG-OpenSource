Welcome to Quadrant Time

Quadrant Time (XXSG) is a local-first PWA productivity tool built on the Eisenhower Matrix. It integrates quadrant-based task management, habit tracking, Pomodoro focus, calendar scheduling, and AI analysis — helping you prioritize what truly matters.

Data is stored locally in your browser. No registration required to start using it right away. Login is available for custom settings; AI features require your own API key.

Quick Start Guide

1

Create a Task

In the "Add Task" tab, enter a title, choose priority (1–4 = four quadrants), and optionally set start/due dates. The system auto-categorizes it.

2

Quadrant View

Switch to "Four Quadrants" tab. Drag tasks across quadrants to change priority. Click a title to edit details.

3

Complete a Task

Click the circle button to toggle completion. Completed tasks hide from quadrants (visible in the dashboard).

4

Edit & Detail

Click the task title to open a detail modal. Edit fields, add subtasks, duplicate, delete, or pin. Drag the modal to reposition it.

5

Reminders & Overdue

Tasks with start dates trigger a popup 5 minutes before. Overdue tasks show a red card with a red left border. Dismiss once, never bothered again.

6

View Switching

Toggle between grid mode and list mode in the quadrant view. Use the expand button in any quadrant header for full-screen focus.

7

Backup Data

Go to Settings or Data Sync page to export JSON. Restore by uploading a backup file. Always export before switching browsers.

8

AI Setup

Admin dashboard → AI Configuration, choose DeepSeek, OpenAI, Claude, Kimi, Qwen, GLM/Z.ai, or MiniMax, then enter the corresponding API Key to use smart task analysis and Daily Fortune AI features.

FAQ

Q1

Will I lose my data?

A: Data is stored in browser localStorage. Clearing browser data, switching browsers, or reinstalling the OS will erase it. Always export JSON backups regularly.

Q2

Can I sync across devices?

A: Not yet — this is a local-first app. For now, export JSON on device A and import it on device B. A cloud sync backend is under consideration.

Q3

How do I use AI features?

A: Go to Admin dashboard → AI Configuration and enter your own DeepSeek, OpenAI, Claude, Kimi, Qwen, GLM/Z.ai, or MiniMax API Key. Daily Fortune AI and AI task analysis share this configuration. Third-party API usage may incur fees from the respective platform.

Q4

How to back up data?

A: Settings → Export Data → JSON format. Restore via the Data Sync page. Export before any major operation.

Q5

Offline capable?

A: Core features (tasks, habits, Pomodoro, dashboard) work offline. AI features need a network connection. Install as PWA for the best offline experience.

Q6

Popup reminders not showing?

A: ① Check notification permission in Settings; ② The reminder polls every 2 seconds; browsers throttle background tabs; ③ Run window.checkReminderSystemStatus() in the console to diagnose.

Q7

How to delete all data?

A: Settings → Data Management → Clear All Data. ⚠️ This is irreversible — export a backup first.
